Kong Helm Charts

This is the official Helm Charts repository for installing Kong on Kubernetes.

Setup

$ helm repo add kong https://bdecoste.github.io/kong-helm/
$ helm repo update

# Helm 3
$ helm install kong/kong --generate-name
